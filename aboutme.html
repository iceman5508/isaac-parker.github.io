Isaac Parker is a creative perfectionist. In some ways I put a piece of myself in every work or project I do, regardless of how minimal it might be. I started on my path of creativity in late 2006 thanks to sony. By this I mean that in late 2006, I recieved an original Playstation Portable (psp) from a friend. What this firend did not tell me is that the disk drive was broken an was unable to play games. Once I realized this major limitation, I asked myself "what else can the psp do if not play games?"
It was this fundimental question that led me to my programming path as well as in trouble with my parents as I started asking the same question about every other electronice device in our house.

IEngine

The IEngine is a framwork that exists in two forms. The first of these forms is IEngine for web.This version of the engine is written entierly in PHP currently it is ina working beta phase. The second version of the Engine is IEngine for desktop. This version is written in Java and can be used to create native desktop application and games. This verson is also in a working beta stage. Both Versions of the Engine are primarily being used to teach intoductory to programming for kids and individuals who want to get involved in programming.

Both versions of the update will recieve periodic updates based on error I find or features that I believe are missing. However, user input is highly recommmended and will make the development and advancement of the Engine much more effective.
